رویدادها - کلیدها در پایگیم

در پست قبلی، ما به بررسی کدهای پایه‌ای پرداختیم. در این پست، به بررسی کد جدیدی خواهیم پرداخت که به طور خاص به مدیریت رویدادها و کلیدها در پایگیم اختصاص دارد.

import pygame

در این خط، همانند قبل، کتابخانه pygame وارد می‌شود.

pygame.init()

این خط برای راه‌اندازی تمام ماژول‌های مورد نیاز pygame استفاده می‌شود.

display = pygame.display.set_mode((500, 500))

پنجره گرافیکی با ابعاد ۵۰۰x۵۰۰ پیکسل ایجاد می‌شود.

for event in events:

حلقه‌ای برای پردازش رویدادهای موجود.

if event.type == pygame.QUIT:
game = False

بررسی رویداد بسته شدن پنجره و تغییر وضعیت بازی.

if event.type == pygame.KEYDOWN:

بررسی فشار کلیدها.

if event.key == pygame.K_a: print("a feshar dade shode!")

چاپ پیام مخصوص کلید A.

elif event.key == pygame.K_s: print("s feshar dade shode!")

چاپ پیام مخصوص کلید S.

elif event.key == pygame.K_d: print("d feshar dade shode!")

چاپ پیام مخصوص کلید D.

elif event.key == pygame.K_w: print("w feshar dade shode!")

چاپ پیام مخصوص کلید W.

pygame.display.update()

بروزرسانی صفحه نمایش.
